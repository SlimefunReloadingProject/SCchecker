# 使用方式

1. 你的电脑需要安装了`python3.11`或以上
2. 将全部文件解压到 `plugins/SlimeCustomizer`
3. 运行 `SC检查器.bat` ，就可以开始快速检查了
等待运行完毕后，会显示出一些潜在bug（若有）

> **注意！此检查器不会检查`name`或`lore`！**
>
> **此外，此脚本的粘液物品检查依赖 `Slimefun` 的 `Items.yml`**
>
> **请确保控制台输出的 Bug（若有）皆已修复，本程序才能正常运行！**
